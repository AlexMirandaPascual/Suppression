Each folder is an individual experiment, to run them just run the main.py and you will get the results if you followed the instructions correctly

## How to Install
    Install python 3.12.6 for Windows 11, 10 for other operating systems it has not been tested
## dependencies
    Need  install numpy, pandas and matplotlib compatibles con python 3.12.6 with the following commands
    pip install numpy
    pip install pandas
    pip install matplotlib
